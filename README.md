AI Mini Project: Spam Message Detection

->Project Title
Spam Message Detection using Machine Learning

-> Objective (2–3 lines)
To build an AI-based system that classifies text messages as Spam or Not Spam (Ham) using Machine Learning and Natural Language Processing techniques.

-> Tools & Technologies
Python
Scikit-learn
Pandas
TF-IDF Vectorizer
Naive Bayes Algorithm

-> Dataset
SMS Spam Collection Dataset
Columns:
label → spam / ham
message → text message
